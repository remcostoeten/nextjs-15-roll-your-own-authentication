# Project Architecture Documentation

## Overview

This project is a modern web application built with Next.js 14, using the App Router, Server Components, and a robust authentication system. The architecture follows domain-driven design principles with a focus on maintainability and type safety.

## Core Technologies

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Database**: Drizzle ORM
- **Authentication**: Custom JWT-based system
- **State Management**: React Context + Server Actions
