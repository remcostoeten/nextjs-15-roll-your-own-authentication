# Development Workflow Tools

An advanced suite of development workflow automation tools powered by local LLMs (Large Language Models). These tools help streamline common development tasks by providing intelligent assistance for commit messages, API documentation, and code optimization.

## Features Overview

### 1. Smart Commit Generator

Automatically generates meaningful commit messages by analyzing your staged changes.

**Key Features:**

-   Analyzes git diffs and staged files
-   Follows conventional commits format
-   Detects breaking changes
-   Provides comprehensive commit descriptions
-   Intelligently determines commit type (feat/fix/refactor)

### 2. API Spec Generator

Automatically generates OpenAPI/Swagger specifications from your route handlers.

**Key Features:**

-   Parses route files for endpoints
-   Extracts request/response models
-   Generates comprehensive API documentation
-   Includes authentication requirements
-   Documents error scenarios

### 3. Code Optimizer

Analyzes code for optimization opportunities and suggests improvements.

**Key Features:**

-   Performance optimization suggestions
-   Code complexity analysis
-   Memory usage recommendations
-   Best practices enforcement
-   Detailed improvement examples

## Prerequisites

-   Python 3.8+
-   A running local LLM server (Ollama, LocalAI, etc.)
-   Git (for commit message generator)

## Installation

```bash
git clone https://github.com/yourusername/dev-workflow-tools
cd dev-workflow-tools
pip install -r requirements.txt
```
